# Zeek XDP Loader

This is an XDP loader for Zeek. This is an entirely standalone component and it owns the XDP programs for built-in XDP as well as the means to manipulate and load them.

For more information about why this is a separate binary, or about why a custom binary is needed in the first place, see the [Zeek discussion](https://github.com/zeek/zeek/discussions/5413) on XDP loading.
